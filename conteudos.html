Conteúdos Estudados

Os principais tópicos aprendidos na disciplina

Durante a disciplina estudamos os fundamentos do HTML, do CSS e uma introdução ao JavaScript. Clique em cada tópico abaixo para ver uma breve explicação sobre ele.

Estrutura básica do HTML
Todo documento HTML fica entre as tags <html> e possui um cabeçalho (<head>, com o <title>) e um corpo (<body>), onde fica o conteúdo principal da página.
Formatação de texto
Usamos tags de título <h1> a <h6>, parágrafos <p> e estilos como negrito (<b>), itálico (<i>) e sublinhado (<u>) para organizar o texto.
Listas
Listas ordenadas (<ol>) mostram itens numerados e listas não ordenadas (<ul>) usam marcadores. Cada item é definido pela tag <li>.
Tabelas
Criadas com <table>, organizam dados em linhas (<tr>) e células de cabeçalho (<th>) ou de dados (<td>).
Links e imagens
Links são feitos com a tag <a href="..."> e imagens com <img src="...">, que permitem interligar páginas e exibir figuras no site.
Formulários
O elemento <form> coleta dados do usuário usando campos como <input>, <textarea> e <select>, além de botões de enviar e limpar.
CSS: sintaxe e tipos
Uma regra CSS segue o formato seletor { atributo: valor; }. O CSS pode ser externo (arquivo .css), interno (tag <style>) ou inline (atributo style).
CSS: margin, padding e atalhos
Margin é o espaço por fora do elemento e padding é o espaço por dentro. Atalhos (shorthand), como a propriedade font, permitem definir várias propriedades em uma única linha.
