Sobre a Disciplina

O que é Desenvolvimento Web I e por que ela é importante

O que é a disciplina

Desenvolvimento Web I é a disciplina do curso de Análise e Desenvolvimento de Sistemas que ensina os fundamentos para criar páginas e sites para a internet. Nela aprendemos como a web funciona e como construir páginas que são interpretadas pelo navegador.

A web é estruturada em dois princípios básicos: o HTTP (protocolo de transferência de hipertexto), que permite a navegação ao clicar em links, e o HTML (linguagem de marcação de hipertexto), no qual as páginas são escritas. A partir daí, usamos o CSS para estilizar e o JavaScript para adicionar interatividade.

Por que ela é importante

Saber desenvolver para a web é uma das habilidades mais requisitadas na área de tecnologia. Praticamente todo sistema hoje tem uma interface acessada pelo navegador, então dominar HTML, CSS e JavaScript é a base para criar desde sites simples até aplicações completas.

O uso de CSS, por exemplo, traz vários benefícios para o desenvolvimento:

  • Economiza tempo de trabalho;
  • Diminui o tamanho do código da página;
  • Faz a página carregar mais rapidamente;
  • Facilita a manutenção e as alterações;
  • Dá mais controle sobre o layout da página.

As três tecnologias

HTML

Estrutura o conteúdo da página por meio de tags, como títulos, parágrafos, listas, tabelas e formulários.

CSS

Cuida da aparência: cores, fontes, espaçamentos e layout. Pode ser externo, interno ou inline.

JavaScript

Adiciona comportamento e interatividade, respondendo a eventos como cliques e movimentos do mouse.
