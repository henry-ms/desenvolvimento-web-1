Minha disciplina favorita no curso de Análise e Desenvolvimento de Sistemas

Bem-vindo(a)!

Este site foi desenvolvido como projeto final da disciplina e tem como tema o curso de Análise e Desenvolvimento de Sistemas do Instituto Federal, com foco na disciplina de Desenvolvimento Web I.

Desenvolvimento Web I é a disciplina em que aprendemos a criar páginas para a internet, utilizando as três principais tecnologias da web: HTML para estruturar o conteúdo, CSS para estilizar e JavaScript para adicionar interatividade. O próprio site que você está navegando foi feito com essas tecnologias.

HTML

A linguagem de marcação que estrutura as páginas da web.

CSS

As folhas de estilo que cuidam da aparência e do layout.

JavaScript

A linguagem que deixa as páginas interativas.
